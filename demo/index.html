Basic kgui-single-digit loop demo

  • Continuous clicking on the '+'-button, will change the digit from '9' to '0'.
  • Continuous clicking on the '-'-button, will change the digit from '0' to '9'.
  • On crossing this border, background will change color as demo.

Border crossing detection is possible by listen to the EVENT `digit-boundary-crossed`.

Basic kgui-single-digit min, max demo
